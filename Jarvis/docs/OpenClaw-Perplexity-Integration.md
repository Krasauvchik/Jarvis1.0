# OpenClaw и Perplexity — интеграция в Jarvis

Документ описывает, как идеи и функционал [OpenClaw](https://github.com/openclaw/openclaw) и Perplexity (в т.ч. Computer / Sonar) встраиваются в Jarvis.

---

## 1. OpenClaw — что берём

| Возможность OpenClaw | Статус в Jarvis | Реализация |
|----------------------|-----------------|------------|
| **Локальные LLM (Ollama)** | ✅ Добавлено | В настройках ИИ можно выбрать «Ollama»; запросы идут на локальный или указанный URL (приватность, офлайн). |
| **Несколько бэкендов ИИ** | ✅ Есть | Модели: Gemini, Эвристика, Ollama, Cloud GPT; без привязки к одному вендору. |
| **Навыки / плагины (вкл/выкл)** | ✅ Основа заложена | В настройках — переключатели «календарь», «почта», «глубокий анализ», «голос»; дальше — расширяемый список навыков. |
| **Автономные действия** | 🔶 Частично | Создание задач, календарь, почта через бэкенд; Siri Intents для голоса. Планируется: больше действий по сценариям (как у OpenClaw). |
| **Мультиканалы (Telegram, WhatsApp…)** | 📋 В плане | Отдельный сервис/бот или плагин OpenClaw, который через API Jarvis создаёт/читает задачи. Требует HTTP API или общее хранилище. |
| **Память и контекст** | 🔶 Частично | iCloud синхронизация задач и настроек; контекст ИИ — текущие задачи. Планируется: долгоживущий контекст для советов. |

---

## 2. Perplexity — что берём

| Возможность | Статус в Jarvis | Реализация |
|-------------|-----------------|------------|
| **Sonar API (поиск в интернете)** | 📋 В плане | Вызов Sonar из бэкенда при запросе советов/фактов; подсказки в Аналитике с актуальной информацией. |
| **Computer (автономные сценарии)** | 📋 Идея | Не интеграция с продуктом Computer, а свои сценарии: «разбей цель на задачи», «спланируй неделю» — разбиение на шаги и создание задач в Jarvis. |

---

## 3. Реализовано в коде

### 3.1 Модели ИИ (AIModels.swift)

- **Gemini** — облако (бэкенд).
- **Эвристика** — офлайн, без сети.
- **Ollama** — локальная или своя модель (Ollama API), приватность.
- **Cloud GPT** — зарезервировано под другие облачные API.
- **На устройстве** — зарезервировано под Apple ML / on-device в будущем.

### 3.2 AIManager

- Выбор модели сохраняется в iCloud и UserDefaults.
- **Глубокий анализ**: при отсутствии сети или ошибке сервера возвращается эвристика (офлайн-режим в духе OpenClaw).
- При выборе **Ollama** запросы уходят на `Config.Endpoints.ollama` (по умолчанию `http://localhost:11434/api/generate`); URL можно задать в настройках (если добавить экран).

### 3.3 Настройки (навыки)

В настройках добавлена секция **«Навыки ИИ»** (в духе OpenClaw):

- **Календарь** — вкл/выкл: при выключении вкладка «Календарь» в iOS скрывается.
- **Почта** — вкл/выкл: при выключении вкладка «Почта» скрывается.
- **Глубокий анализ** — вкл/выкл: при выключении блок «Глубокий анализ» в Аналитике не показывается.
- **Голосовой ввод** — вкл/выкл: зарезервировано для будущего отключения голоса в Плане.

Значения хранятся в UserDefaults (`jarvis_skill_*`). По умолчанию все включены.

## 4. Дорожная карта (функционал в духе OpenClaw)

1. **Ollama** — ✅ добавлен как модель; при необходимости — настройка URL в UI.
2. **Навыки в настройках** — список вкл/выкл (календарь, почта, глубокий анализ, голос) и сохранение в UserDefaults/iCloud.
3. **Сценарии** — «Разбей цель на задачи», «Спланируй неделю»: вызов LLM (Gemini/Ollama) и создание задач в PlannerStore.
4. **Sonar** — опциональный вызов Perplexity Sonar из бэкенда для подсказок с поиском в интернете.
5. **API для внешних агентов** — простой REST или WebSocket для добавления/чтения задач (для бота OpenClaw или своих скриптов).
6. **Каналы** — Telegram/WhatsApp-бот, который через API Jarvis работает с задачами; или плагин OpenClaw «Jarvis», который дергает этот API.
